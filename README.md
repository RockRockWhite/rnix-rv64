# rnix-rv64 
## About
基于 [rcore-os](https://github.com/rcore-os/rCore-Tutorial-v3) 的类unix操作系统内核。算是 [rnix-x86](https://github.com/RockRockWhite/rnix-x86) 的延续。

运行ISA：Risc-V 64
